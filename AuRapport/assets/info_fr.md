# Au‑Rapport

Au‑Rapport analyse votre système A.V.A.T.A.R et affiche le **nombre exact de plugins installés**.

## 🎯 Fonction principale
- `au rapport`
- `combien de plugins sont installés`
- `donne-moi le rapport des plugins`

## 🗂️ Utilisation par pièce
- `au rapport dans le Salon`
- `au rapport dans la Cuisine`

## 📄 Notes
Ce plugin est idéal pour vérifier rapidement l’état de votre installation.
